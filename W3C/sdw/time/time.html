Namespace URI

The namespace URI for OWL-Time is http://www.w3.org/2006/time#.

The vocabulary is published both in human- and machine-readable forms. A GET request made to the canonical namespace URI will cause HTTP Content Negotiation to occur, which means that the most appropriate variant will be served according to your request. However, if you explicitly wish to retrieve a particular serialisation, you can also do so using the following URLs:
