The {{domxref(“HTMLVideoElement”)}} disablePictureInPicture property reflects the HTML attribute indicating whether the user agent should suggest the picture-in-picture feature to users, or request it automatically.

Syntax

disablePictureInPicture = htmlVideoElement.disablePictureInPicture;

Value

A boolean value that is true if the user agent should suggest that feature to users.
