The CSSKeywordValue() constructor creates a new {{domxref(“CSSKeywordValue”)}} object which represents CSS keywords and other identifiers.

Syntax

var cssKeywordValue = new CSSKeywordValue(val)

Parameters

  • value
    • : Sets or returns the value of the new CSSKeywordValue.

Exceptions

  • TypeError
    • : If the value parameter is not specified or it is not of type {{jsxref(‘String’)}}.

Examples

The following example resets the CSS {{cssxref(‘display’)}} property to its defaults, setting the inline style attribute to style="display: initial" if viewed in the developer tools inspector.

css hidden #myElement { display: flex; }

html hidden <div id="myElement">Check the developer tools to see the log in the console and to inspect the style attribute on this div.</div>

let keyword = new CSSKeywordValue('initial');
let myElement = document.getElementById('myElement').attributeStyleMap;
    myElement.set('display', keyword);

console.log( myElement.get('display').value);  // 'initial'
console.dir( keyword );
