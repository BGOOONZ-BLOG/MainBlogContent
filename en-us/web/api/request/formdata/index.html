{{APIRef(“Fetch”)}}

The formData() method of the {{domxref(“Request”)}} interface reads the request body and returns it as a promise that resolves with a {{domxref(“FormData”)}} object.

Syntax

request.formData().then(function(formdata) {
  // do something with your formdata
});

Parameters

None.

Return value

A {{jsxref(“Promise”)}} that resolves with a {{domxref(“FormData”)}} object.

Examples

const formData = new FormData();
const fileField = document.querySelector('input[type="file"]');

formData.append('username', 'abc123');
formData.append('avatar', fileField.files[0]);

const request = new Request('/myEndpoint', {
  method: 'POST',
  body: formData
});

request.formData().then(function(data) {
  // do something with the formdata sent in the request
});
