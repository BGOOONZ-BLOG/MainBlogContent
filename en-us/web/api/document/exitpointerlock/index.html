The exitPointerLock() method asynchronously releases a pointer lock previously requested through {{domxref(“Element.requestPointerLock”)}}.

To track the success or failure of the request, it is necessary to listen for the {{event(“pointerlockchange”)}} and {{event(“pointerlockerror”)}} events.

Syntax

document.exitPointerLock();
