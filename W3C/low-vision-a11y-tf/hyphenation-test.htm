Hyphenation hints (&shy; or &#00AD;) work with IE, Chrome, Safari and no css hyphens declaration.

Hyphenation hints (&shy; or &#00AD;) work with Firefox, IE, Chrome, Safari and css hyphens:manual declaration.

CSS hyphens: auto (with browser prefixes) work with Firefox, IE, and Safari.
