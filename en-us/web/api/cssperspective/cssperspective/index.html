The CSSPerspective() constructor creates a new {{domxref(“CSSPerspective”)}} object representing the perspective() value of the individual {{CSSXref(‘transform’)}} property in CSS.

Syntax

var CSSPerspective = new CSSPerspective(length);

Parameters

  • {{domxref(‘CSSPerspective.length’,‘length’)}}
    • : A value for the distance from z=0 of the {{domxref(‘CSSPerspective’)}} object to be constructed. This must be a {{cssxref(‘length’)}}.

Exceptions

  • TypeError
    • : Raised if the value of CSSPerspective.length exists but is not a {{cssxref(‘length’)}}.
