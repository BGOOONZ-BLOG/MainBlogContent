{{APIRef(“DOM Events”)}}

The InputEvent.isComposing read-only property returns a boolean value indicating if the event is fired after {{event(“compositionstart”)}} and before {{event(“compositionend”)}}.

Syntax

var bool = event.isComposing;

Example

var inputEvent = new InputEvent('syntheticInput', false);
console.log(inputEvent.isComposing); // return false
