{{APIRef}}

The pointerout event is fired for several reasons including: pointing device is moved out of the hit test boundaries of an element; firing the {{event(“pointerup”)}} event for a device that does not support hover (see {{event(“pointerup”)}}); after firing the {{event(“pointercancel”)}} event (see {{event(“pointercancel”)}}); when a pen stylus leaves the hover range detectable by the digitizer.

Bubbles Yes
Cancelable Yes
Interface {{domxref(“PointerEvent”)}}
Event handler property onpointerout

Examples

Using addEventListener():

const para = document.querySelector('p');

para.addEventListener('pointerout', (event) => {
  console.log('Pointer moved out');
});

Using the onpointerout event handler property:

const para = document.querySelector('p');

para.onpointerout = (event) => {
  console.log('Pointer moved out');
};
