The onmousewheel property sets and returns the event handler for the {{event(“mousewheel”)}} event.

Instead of using this event, use the standard {{event(“wheel”)}} event.

Syntax

element.onmousewheel = handlerFunction;
var handlerFunction = element.onmousewheel;

handlerFunction should be either null or a JavaScript function specifying the handler for the event.

Notes

See the DOM event handlers page for information on working with on... handlers.

The mousewheel event is fired asynchronously when a mouse wheel or similar device is operated. It’s represented by the WheelEvent interface.

See the {{event(“mousewheel”)}} event documentation for more information about the event.
