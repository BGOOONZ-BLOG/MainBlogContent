Open-source Projects

The community maintains a number of open-source projects including this documentation.

testtwf-website

  Repository
Contributor's Guide
Creative Commons Attribution License (CC-BY 3.0)

testtwf-website contains the source code of this very website which includes all documentation pertaining to the Open Web Platform testing effort.

web-platform-tests

  Repository
Contributor's Guide
W3C Test Suite License

web-platform-tests is the main test repository for the Open Web Platform. It contains all the existing test suites for Open Web Platform sepecifications excpet for those of the CSS Working Group.

testharness.js

  Repository
W3C Test Suite License

testharness.js is the JavaScript testing framework used at W3C. This repository also contains idlharness.js and WebIDLParser.js which are used to automatically generate tests from Web IDL snippets contained in specs.
