The getDevices() method of {{DOMxRef(“Bluetooth”)}} interface of Web Bluetooth API exposes the Bluetooth devices this origin is allowed to access. This method does not display any permission prompts.

Note: This method returns a {{DOMxRef(“BluetoothDevice”)}} for each device the origin is currently allowed to access, even the ones that are out of range or powered off.

Syntax

var readerPromise = Bluetooth.getDevices();

Parameters

None.

Return value

A {{JSxRef(“Promise”)}} that resolves with an array of {{DOMxRef(“BluetoothDevice”)}}s.

Exceptions

This method doesn’t throw any exceptions.
