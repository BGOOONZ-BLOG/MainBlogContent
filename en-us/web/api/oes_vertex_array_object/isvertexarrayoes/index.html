The OES_vertex_array_object.isVertexArrayOES() method of the WebGL API returns true if the passed object is a {{domxref(“WebGLVertexArrayObject”)}} object.

Syntax

GLBoolean ext.isVertexArrayOES(arrayObject);

Parameters

  • arrayObject
    • : A {{domxref(“WebGLVertexArrayObject”)}} (VAO) object to test.

Return value

A {{domxref(“WebGL_API.Types”)}} indicating whether the given object is a {{domxref(“WebGLVertexArrayObject”)}} object (true) or not (false).

Examples

var ext = gl.getExtension('OES_vertex_array_object');
var vao = ext.createVertexArrayOES();
ext.bindVertexArrayOES(vao);

// ...

ext.isVertexArrayOES(vao);
