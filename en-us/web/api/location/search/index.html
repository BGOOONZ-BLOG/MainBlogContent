{{ApiRef(“Location”)}}

The search property of the {{domxref(“Location”)}} interface is a search string, also called a query string; that is, a {{domxref(“USVString”)}} containing a '?' followed by the parameters of the URL.

Modern browsers provide URLSearchParams and URL.searchParams to make it easy to parse out the parameters from the querystring.

Syntax

string = object.search;
object.search = string;

Examples

// Let an <a id="myAnchor" href="/en-US/docs/Location.search?q=123"> element be in the document
var anchor = document.getElementById("myAnchor");
var queryString = anchor.search; // Returns:'?q=123'

// Further parsing:
let params = new URLSearchParams(queryString);
let q = parseInt(params.get("q")); // is the number 123
