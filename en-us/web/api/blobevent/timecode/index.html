The timecode readonlyinline property of the {{domxref(“BlobEvent”)}} interface a {{domxref(“DOMHighResTimeStamp”)}} indicating the difference between the timestamp of the first chunk in data, and the timestamp of the first chunk in the first BlobEvent produced by this recorder.

Note that the timecode in the first produced BlobEvent does not need to be zero.

Syntax

var timecode = blobEvent.timecode

Value

A {{domxref(“DOMHighResTimeStamp”)}}.
