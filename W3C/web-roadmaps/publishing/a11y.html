Accessibility technologies necessary and influenced by publishing

Accessibility plays a very important role in publishing, raising specific concerns and requirements.

  • Web Accessibility Guidelines (WCAG) 2.1 is under development at W3C; this version may be influenced by the specific requirements of publishing.
  • WAI-ARIA 1.1 is at the basis of the publishing extension of ARIA (DPUB-ARIA)
  • Core Accessibility API Mappings is the core set of accessibility mappings of the ARIA to platform accessibility API-s
  • Digital Publishing WAI-ARIA Module 1.0 is the first publishing specific ARIA module, which serves at the bases of a more extended version to be developed at the Publishing Working Group.
  • Digital Publishing Accessibility API Mappings is the set of accessibility mappings of the DPUB WAI-ARIA module to platform accessibility API-s
