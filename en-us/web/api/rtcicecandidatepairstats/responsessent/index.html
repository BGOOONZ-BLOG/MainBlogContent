The {{domxref(“RTCIceCandidatePairStats”)}} dictionary’s responsesSent property indicates the total number of {{Glossary(“STUN”)}} connectivity check responses that have been sent so far on the connection described by this pair of candidates.

Syntax

responsesSent = rtcIceCandidatePairStats.responsesSent;

Value

An integer value indicating the number of times a response has been sent to a {{Glossary(“STUN”)}} connectivity check request.

Note: Since it isn’t possible to tell the difference between connectivity check requests and consent requests, this value includes both.
