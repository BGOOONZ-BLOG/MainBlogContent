The sync property of the {{domxref(“ServiceWorkerRegistration”)}} interface returns a reference to the {{domxref(“SyncManager”)}} interface, which manages background synchronization processes.

Syntax

var syncManager = serviceWorkerRegistration.sync;

Value

A {{domxref(“SyncManager”)}} object.
