The PaymentRequestUpdateEvent() constructor creates a new {{domxref(“PaymentRequestUpdateEvent”)}} object which enables a web page to update the details of a {{domxref(“PaymentRequest”)}} in response to a user action. Actual updates are made by passing options to the {{domxref(‘PaymentRequestUpdateEvent.updateWith’,‘updateWith()’)}} method.

Syntax

var paymentRequestUpdateEvent = new PaymentRequestUpdateEvent()

Parameters

None.

Return value

A new PaymentRequestUpdateEvent
