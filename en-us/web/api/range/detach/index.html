{{ApiRef(“DOM”)}}

The Range.detach() method does nothing. It used to disable the {{domxref(“Range”)}} object and enable the browser to release associated resources. The method has been kept for compatibility.

Syntax

range.detach();

Example

var range = document.createRange();

range.selectNode(document.getElementsByTagName("div").item(0));
range.detach();
