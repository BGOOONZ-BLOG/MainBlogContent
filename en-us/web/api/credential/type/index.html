The type property of the {{domxref(“Credential”)}} interface returns a {{domxref(“DOMString”)}} containing the credential’s type. Valid values are password, federated and public-key.

Syntax

var credType = Credential.type;

Value

A {{domxref(“DOMString”)}} contains a credential’s given name.
