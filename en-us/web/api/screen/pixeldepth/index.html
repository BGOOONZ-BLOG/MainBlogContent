{{APIRef(“CSSOM”)}}

Returns the bit depth of the screen. Per the CSSOM, some implementations return 24 for compatibility reasons. See the {{anch(“Browser compatibility”, “browser compatibility”)}} section for those that don’t.

Syntax

let depth = window.screen.pixelDepth

Example

// if there is not adequate bit depth
// choose a simpler color
if ( window.screen.pixelDepth > 8 ) {
  document.style.color = "#FAEBD7";
} else {
  document.style.color = "#FFFFFF";
}
