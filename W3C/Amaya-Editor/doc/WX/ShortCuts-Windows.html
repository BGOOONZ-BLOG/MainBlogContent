Amaya

Amaya keyboard shortcuts for Windows

Amaya defines two kinds of keyboard shortcuts for Windows: shortcuts using standard Windows keys (ex. Ctrl+C to copy the selection) and shortcuts using sequences (ex. Ctrl-t Ctrl-t to create a table).
