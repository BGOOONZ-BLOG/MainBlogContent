The as property of the {{domxref(“HTMLLinkElement”)}} interface returns a {{domxref(“DOMString”)}} representing the type of content being loaded by the HTML link, one of "script", "style", "image", "video", "audio", "track", "font", "fetch".

Syntax

var as = HTMLLinkElement.as
HTMLLinkElement.as = as

Value

A {{domxref(“DOMString”)}}.
