Test Suite for the Time Ontology in OWL (OWL-Time)

RDF resources that may be used to test conformance to the rules for OWL-Time specified in:

  • Time Ontology in OWL (OWL-Time) are available in https://github.com/w3c/sdw/tree/gh-pages/time/test-suite
  • Extensions to the OWL-Time Ontology - entity relations are available in https://github.com/w3c/sdw/tree/gh-pages/time-entity-relations/test-suite

Status of this document

The test suite is published by the Spatial Data on the Web Interest Group.
