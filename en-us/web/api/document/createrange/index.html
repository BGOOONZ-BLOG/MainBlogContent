{{APIRef(“DOM”)}}

The Document.createRange() method returns a new {{domxref(“Range”)}} object.

Syntax

range = document.createRange();

range is the created {{domxref(“Range”)}} object.

Example

let range = document.createRange();

range.setStart(startNode, startOffset);
range.setEnd(endNode, endOffset);

Notes

Once a Range is created, you need to set its boundary points before you can make use of most of its methods.
