{{APIRef(“DOM Events”)}} {{Deprecated_Header}}

The cancelBubble property of the {{domxref(“Event”)}} interface is deprecated. Use {{domxref(“Event.stopPropagation()”)}} instead. Setting its value to true before returning from an event handler prevents propagation of the event. In later implementations, setting this to false does nothing. See {{anch(“Browser compatibility”)}} for details.

Value

A boolean value. The value true means that the event must not be propagated further.

Example

elem.onclick = function(event) {
  // Do cool things here
  event.cancelBubble = true;
}
