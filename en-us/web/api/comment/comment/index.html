The Comment() constructor returns a newly created {{domxref(“Comment”)}} object with the optional string given in parameter as its textual content.

Syntax

new Comment();
new Comment(aString);

Parameters

  • aString {{optional_inline}}

Return value

A new {{domxref(“Comment”)}} containing aString, or the empty string if no parameter was given.

Example

let comment = new Comment("Test");
