{{APIRef(“HTML DOM”)}}

The HTMLMediaElement.controls property reflects the {{htmlattrxref(“controls”, “video”)}} HTML attribute, which controls whether user interface controls for playing the media item will be displayed.

Syntax

var ctrls = video.controls;
audio.controls = true;

Value

A boolean value. A value of true means controls will be displayed.

Example

var obj = document.createElement('video');
obj.controls = true;
