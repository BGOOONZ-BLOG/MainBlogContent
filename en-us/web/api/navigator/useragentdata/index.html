The userAgentData read-only property of the {{domxref(“Navigator”)}} interface returns a {{domxref(“NavigatorUAData”)}} object which can be used to access the {{domxref(“User-Agent Client Hints API”)}}.

Syntax

let userAgentData = navigator.userAgentData

Value

A {{domxref(“NavigatorUAData”)}} object.

Examples

The following example prints the value of {{domxref(“NavigatorUAData.brands”)}} to the console.

console.log(navigator.userAgentData.brands);
