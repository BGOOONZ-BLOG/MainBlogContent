The repetitionCount property of the {{domxref(“ImageTrack”)}} interface returns the number of repetitions of this track. ### Value

An integer.

Examples

The following example prints the value of repetitionCount to the console.

let track = imageDecoder.tracks.selectedTrack;
console.log(track.repetitionCount);
