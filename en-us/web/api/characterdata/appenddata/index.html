The appendData() method of the {{domxref(“CharacterData”)}} interface adds the provided data to the end of the node’s current data.

Syntax

appendData(data);

Parameters

  • data
    • : The data to append to the current node.

Return value

None.

Example

<span>Result: </span>A text
let span = document.getElementsByTagName("span")[0];
let textnode = span.nextSibling;

textnode.appendData(" - appended text.");
