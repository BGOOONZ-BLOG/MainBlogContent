The Navigator.mediaDevices read-only property returns a {{domxref(“MediaDevices”)}} object, which provides access to connected media input devices like cameras and microphones, as well as screen sharing.

Syntax

var mediaDevices = navigator.mediaDevices;

Return value

The {{domxref(“MediaDevices”)}} singleton object. Usually, you just use this object’s members directly, such as by calling {{domxref(“MediaDevices.getUserMedia”, “navigator.mediaDevices.getUserMedia()”)}}.
