The withCredentials read-only property of the {{domxref(“EventSource”)}} interface returns a boolean value indicating whether the EventSource object was instantiated with CORS credentials set.

Syntax

var myWithCredentials = eventSource.withCredentials;

Value

A boolean value indicating whether the EventSource object was instantiated with CORS credentials set (true), or not (false, the default).

Examples

var evtSource = new EventSource('sse.php');
console.log(evtSource.withCredentials);

Note: You can find a full example on GitHub — see Simple SSE demo using PHP.
