The HTMLSelectElement.checkValidity() method checks whether the element has any constraints and whether it satisfies them. If the element fails its constraints, the browser fires a cancelable {{domxref(“HTMLSelectElement/invalid_event”, “invalid”)}} event at the element, and then returns false.

Syntax

var result = selectElt.checkValidity();
