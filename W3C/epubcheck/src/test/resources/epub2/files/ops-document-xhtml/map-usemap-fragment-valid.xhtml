XHTML 1.1 ImageMap example

This is an example of an ImageMap in XHTML 1.1 (EPUB 2).

After the latest revision of the XHTML 1.1 DTD in 2010 the @usemap attribute has now datatype URIREF (basically CDATA) instead of IDREF which allows both of the following map referencing examples whereas the first example with a leading # is to be prefered

At the time of this writing, oXygen will report the first example as invalid because they also use an outdated version of the XHTML 1.1 DTD for validation. However, this will be fixed in a future release.
