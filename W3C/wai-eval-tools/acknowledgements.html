Acknowledgements for Web Accessibility Evaluation Tools List

Back to the Evaluations Tools List

The List of Web Accessibility Evaluation Tools was initially developed in 2001 by Wendy Chisholm and Judy Brewer. Since 2003, it has been managed by Shadi Abou-Zahra. The user interface is developed through the Education and Outreach Working Group (EOWG); the content is managed with help of the Evaluation and Repair Tools Working Group (ERT WG). Over the years there have been many contributors.
