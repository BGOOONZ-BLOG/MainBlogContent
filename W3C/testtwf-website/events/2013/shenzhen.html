关于 | About

Test the Web Forward将引领你更近距离接触你平时所用来创作Web体验的技术规范。在此次活动中，来自于W3C，Adobe，Intel，Mozilla， Google和Opera的专家们及众多独立开发者将手把手的引导你逐步学习阅读并理解相关规范，并且为这些规范创建和贡献全新的测试用例。你还可以了解到最新的HTML5和CSS3规范的最新状态并且获悉为何测试让浏览器一致实现这些新功能的迫切性。通过参加此次活动，你会正式投入一场塑造未来Web技术的运动中！。

Test the Web Forward brings you closer to the very specifications that you use to craft web experiences. At this event, experts from the W3C, Adobe, Intel, Mozilla, Google, and Opera as well as independent developers will be on hand to guide you step-by-step through reading & understanding specs and through creating & contributing new tests for them. You'll also learn the status of latest HTML5 and CSS3 specifications and why there is an urgent need for tests to make browser implementations of these features consistent. By participating in this event, you are joining a movement that is shaping the future of the web.

参会要求 | Requirements

可熟练使用HTML, JavaScript和CSS编程。本次活动请自带笔记本。

在活动正式开始前

请完成如下的准备工作

  • GitHub的设置
  • 选择一款你喜欢的文本编辑器或IDE。
  • 安装各主流浏览器(如Firefox、Chrome、Opera等)
  • 下载 testharness.js

如果你有问题，我们有专家会帮助你

在参与活动前你需要阅读和熟悉以下至少一个功能，这样会让你有些基础。以下是我们将着重学习和测试的web功能:

HTML5:

  • HTML5 Forms

CSS:

  • CSS Basic User Interface
  • CSS Shapes
  • CSS Masking

FXTF:

  • Compositing & Blending

Browser and Testing Tools:

  • WebDriver

Accessibility:

  • Accessibility Testing

强烈建议您可以提前选择好您感兴趣的W3C规范

You must be able to hand code HTML, JavaScript and CSS to participate. Please bring your own laptop.

Before you begin

Please complete the following in preparation for the event:

  • GitHub Setup
  • Choose your favorite text editor or IDE
  • Install all major browsers (such as Chrome, Firefox, Internet Explorer, or Opera)
  • Be sure you have testharness.js

If you have problems with setup, experts can help you at the event.

Prior to attending, you will want to read and become familiar with at least one of the specs so that you have the foundation required to participate.

Here are the specs we will focus on learning and testing:

HTML5:

  • HTML5 Forms

CSS:

  • CSS Basic User Interface
  • CSS Shapes
  • CSS Masking

FXTF:

  • Compositing & Blending

Browser and Testing Tools:

  • WebDriver

Accessibility:

  • Accessibility Testing

You may also choose any other W3C spec that interests you.
