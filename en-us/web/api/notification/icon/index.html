The icon read-only property of the {{domxref(“Notification”)}} interface contains the URL of an icon to be displayed as part of the notification, as specified in the icon option of the {{domxref(“Notification.Notification”,“Notification()”)}} constructor.

Syntax

var icon = Notification.icon;

Value

A {{domxref(“USVString”)}}.

Examples

In our To-do list app (view the app running live), we use the {{domxref(“Notification.Notification”,“Notification()”)}} constructor to fire a notification, passing it arguments to specify the body, icon and title we want.

var notification = new Notification('To do list', {
  body: text,
  icon: img
});
