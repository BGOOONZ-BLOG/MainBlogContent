The read-only **’shiftKey** property of theTouchEventinterface returns a boolean value indicating whether or not the <kbd>shift</kbd> key is enabled when the touch event is created. If this key is enabled, the attribute's value istrue. Otherwise, it isfalse`.

Syntax

var shiftEnabled = touchEvent.shiftKey;

Return value

The boolean value true if the shift key is enabled for this event; and false if the shift key is not enabled.

Example

The TouchEvent.altKey example includes an example of this property’s usage.
