The get property of the {{domxref(“MediaKeyStatusMap”)}} interface returns the value associated with the given key, or undefined if there is none.

Syntax

var value = mediaKeyStatusMap.get(key);

Parameters

  • key
    • : The key whose value you want returned.

Returns

The value associated with the given key, or undefined.
