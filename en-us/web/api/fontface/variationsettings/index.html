{{APIRef(“CSS Font Loading API”)}}

The variationSettings property of the {{domxref(“FontFace”)}} interface retrieves or sets low-level OpenType or TrueType font variations. It is equivalent to the {{cssxref(“@font-face/font-variation-settings”, “font-variation-settings”)}} descriptor.

Syntax

let variationSettingDescriptor = FontFace.variationSettings;
FontFace.variationSettings = variationSettingDescriptor;

Value

A {{domxref(‘CSSOMString’)}} containing a descriptor.
