The SyncEvent.lastChance read-only property of the {{domxref(“SyncEvent”)}} interface returns true if the user agent will not make further synchronization attempts after the current attempt. This is the value passed in the lastChance parameter of the {{domxref(“SyncEvent.SyncEvent”,“SyncEvent()”)}} constructor.

Syntax

var lastChance = SyncEvent.lastChance

Value

A boolean value that indicates whether the user agent will not make further synchronization attempts after the current attempt.
