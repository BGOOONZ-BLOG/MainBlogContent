The Navigator.hid read-only property returns an {{domxref(“HID”)}} object providing methods for connecting to HID devices, listing attached HID devices, and event handlers for connected HID devices.

Syntax

var hidVar = navigator.hid

Value

An {{domxref(“HID”)}} object.
