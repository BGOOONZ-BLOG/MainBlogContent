The MediaKeySession.close() method notifies that the current media session is no longer needed, and that the content decryption module should release any resources associated with this object and close it. Then, it returns a {{jsxref(‘Promise’)}}.

Syntax

mediaKeySession.close().then(function() { /* ... */ });

Return value

A {{jsxref(‘Promise’)}}.
