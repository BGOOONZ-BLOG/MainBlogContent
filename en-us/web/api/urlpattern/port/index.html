{{ APIRef(“URL Pattern API”) }}

The port property of the {{domxref(“URLPattern”)}} interface is a {{domxref(“USVString”)}} containing the pattern used to match the port part of a URL. This value may differ from the input to the constructor due to normalization.

{{AvailableInWorkers}}

Value

A {{domxref(“USVString”)}}.

Examples

The below example creates a {{domxref(“URLPattern”)}} object with (80|443|8080) for the port part. This pattern matches the port 80, 443, and 8080.

const pattern = new URLPattern({ port: '(80|443|8080)' });
console.log(pattern.port); // '(80|443|8080)'
