{{APIRef(“Shadow DOM”)}}

The innerHTML property of the {{domxref(“ShadowRoot”)}} interface sets or returns a reference to the DOM tree inside the ShadowRoot.

Syntax

var domString = shadowRoot.innerHTML
shadowRoot.innerHTML = domString

Value

A {{domxref(“DOMString”)}}.

Examples

let customElem = document.querySelector('my-shadow-dom-element');
let shadow = customElem.shadowRoot;

shadow.innerHTML = '<strong>This element should be more important!</strong>';
