The SVGAnimatedLength interface represents attributes of type <length> which can be animated.

Properties

  • {{domxref(“baseVal”)}} {{ReadOnlyInline}}
    • : A {{domxref(“SVGLength”)}} representing the base value of the given attribute before applying any animations.
  • {{domxref(“animVal”)}} {{ReadOnlyInline}}
    • : If the given attribute or property is being animated, a {{domxref(“SVGLength”)}} containing the current animated value of the attribute or property. If the given attribute or property is not currently being animated, a {{domxref(“SVGLength”)}} containing the same value as baseVal.

Methods

This interface does not implement any specific methods.
