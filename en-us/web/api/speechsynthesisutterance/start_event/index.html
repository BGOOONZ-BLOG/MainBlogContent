The start event of the Web Speech API {{domxref(“SpeechSynthesisUtterance”)}} object is fired when the utterance has begun to be spoken.

Bubbles No
Cancelable No
Interface {{domxref(“SpeechSynthesisEvent”)}}
Event handler property onstart

Examples

You can use the start event in an addEventListener method:

utterThis.addEventListener('start', function(event) {
  console.log('We have started uttering this speech: ' + event.utterance.text);
});

Or use the onstart event handler property:

utterThis.onstart = function(event) {
  console.log('We have started uttering this speech: ' + event.utterance.text);
}
