The kind readonly property of the {{domxref(“MediaDeviceInfo”)}} interface returns an enumerated value, that is either “videoinput”, “audioinput” or “audiooutput”.

Syntax

var kind = MediaDeviceInfo.kind

Value

A {{domxref(“DOMString”)}}.
