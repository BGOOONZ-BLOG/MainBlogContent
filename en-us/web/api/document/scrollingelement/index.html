The scrollingElement read-only property of the {{domxref(“Document”)}} interface returns a reference to the {{domxref(“Element”)}} that scrolls the document. In standards mode, this is the root element of the document, {{domxref(“document.documentElement”)}}.

When in quirks mode, the scrollingElement attribute returns the HTML body element if it exists and is potentially scrollable, otherwise it returns null.

Syntax

var element = document.scrollingElement;

Example

var scrollElm = document.scrollingElement;
scrollElm.scrollTop = 0;
