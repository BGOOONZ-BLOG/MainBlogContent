Summary

The fileHandle property gives access to the {{domxref(“IDBMutableFile”)}} object that produced the {{domxref(“LockedFile”)}} object.

Syntax

var handler = instanceOfLockedFile.fileHandle

Value

A {{domxref(“IDBMutableFile”)}} object.
