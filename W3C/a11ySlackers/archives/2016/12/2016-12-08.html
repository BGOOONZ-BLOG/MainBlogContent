  24. James Nurthen
    @jnurthen
    Dec 08 20:50
    aria-required is only allowed on
    combobox
    gridcell
    listbox
    radiogroup
    spinbutton
    textbox
    tree
  25. fieldset doesn't map to any of those
  26. https://www.w3.org/TR/html-aam-1.0/#mapping_general
  27. if you want to do that you would need to add a role=radiogroup on the fieldset as you suspect
  28. (not saying it will change the bahviour at all - but to comply with the spec this is necessary)
  29. VO is particularly picky on things like this and will ignore invalid attributes
