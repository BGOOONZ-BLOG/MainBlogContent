{{APIRef(“DOM”)}}

The head read-only property of the {{domxref(“Document”)}} interface returns the {{HTMLElement(“head”)}} element of the current document.

Syntax

var objRef = document.head;

Value

An {{domxref(“HTMLHeadElement”)}}.

Example

<!doctype html>
<head id="my-document-head">
  <title>Example: using document.head</title>
</head>

<script>
  var theHead = document.head;

  console.log(theHead.id); // "my-document-head";

  console.log( theHead === document.querySelector("head") ); // true
</script>

Notes

document.head is read-only. Trying to assign a value to this property will fail silently or, in Strict Mode, throws a {{jsxref(“TypeError”)}} .
