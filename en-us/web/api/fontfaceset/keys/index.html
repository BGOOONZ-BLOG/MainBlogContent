The keys() method of the {{domxref(“FontFaceSet”)}} interface is an alias for {{domxref(“FontFaceSet.values”)}}.

Syntax

FontFaceSet.keys()

Return Value

A new iterator object containing the values for each element in the given FontFaceSet, in insertion order.
