행사 소개 | About

국내 웹 개발자 커뮤니티 멤버 및 해외에서 방문하는 웹 표준 전문가들과 함께 웹 표준에 대한 이해를 높히고, 직접 테스트해 보는 즐거운 교류 행사로서 더 나은 웹 상호 운용성을 위해 웹 표준에 직접 구체적인 공헌을 할 수 있을 기회입니다!

Test The Web Forward은 웹 개발자 커뮤니티가 웹 표준 프로세스에 참여하여 웹을 보다 좋게 만들고자 하는 취지의 행사입니다. 본 행사를 통해 참가자는 표준 웹 활용 환경을 구축하는 데 빠뜨릴 수 없는 웹 표준의 실제 규격을 가까이 접하는 것으로, 웹의 미래를 위하여 무엇을 공헌할 수 있는지를 발견할 수 있을 것입니다. 웹 개발자 및 커뮤니티에서 웹 표준 제정 프로세스에 보다 깊이 관여함으로서 개방 웹을 위한 활동에 참여할 수 있고, 실제 웹 표준 사양에 직접 참여할 수 있습니다. 행사 기간 동안 국내외 웹 표준 전문가들과 웹 표준 스펙에 대해 연구하고 테스트 케이스 제작에 참여할 수 있습니다. 본 행사는 지금까지 샌 프란시스코, 파리, 북경, 시드니, 시애틀 등 세계 각국에서 개최되어 왔고 드디어 한국에서 처음으로 개최하게 되었습니다.

행사 참가자는 HTML, CSS, 및 JavaScript로 코드 경험자를 대상으로 하며, W3C, ETRI, 다음 커뮤니케이션, 네이버, LG전자, Mozilla, Opera에서 참여하는 표준 전문가 및 기술 도우미들과 함께 단계적으로 테스트 작성 방법을 배울 수 있습니다. 본 행사 후 웹 브라우저 내부 구조와, 견고한 JS 테스트를 작성하기 위한 지식 및 향후 W3C 표준 활동에 참여할 수 기반 지식을 배울 수 있습니다.

Come meet members of the Korean Web community and experts from all over the world for a weekend of learning, coding, food, drinks and fun. Help make a more interoperable & better Web!

Test The Web Forward is a movement to make the Web better by getting the Web community involved in Web standards. We bring you closer to the very specifications that you use to craft web experiences and help you discover how you can shape the future of the Web. This is done through education, test contributions and by growing the community. This movement has already rallied the Web communities of San Francisco, Paris, Beijing, Sydney and Seattle, now we are honored to bring it to Seoul!

Over the course of the event, not only will you learn to understand different browsers, but you will also create robust tests along with the editors of various standards to ensure browsers implement these features consistently. At the end of it, you will gain a deeper understanding of browser internals & how you can write clear, robust tests.

We expect you to be able to code in HTML, CSS or JavaScript, but there will be experts from the W3C, ETRI, Daum, Naver, LG Electronics, Mozilla, Opera and others that will guide you step-by-step to create tests.

필요 요건 | Requirements

참석하시는 분은 HTML, 자바 스크립트, CSS 프로그래밍을 할 수 있어야 합니다. 이 행사 참석시에는 자신의 노트북을 가져와주세요.

시작하기 전에

효과적인 행사 진행을 위해 다음과 같은 준비를 해주세요

  • GitHub 가입 및 설정
  • 좋아하는 텍스트 편집기 나 IDE를 선택하여 설치해주세요.
  • 모든 주요 브라우저 설치 (파이어폭스, 크롬, 오페라 등)
  • testharness.js 다운로드

행사를 진행하면서 여러분이 질문이 있는 경우에는 언제든 전문가들이 도움을 드릴 것입니다. 두려워하거나 걱정하지 마세요 ^^

활동에 참여하기 전에 당신은 다음과 같은 규격들을 읽어 보시기 바랍니다. 본 행사에서는 주요 웹 표준 규격의 학습과 테스트에 촛점을 맞출 것입니다.

HTML5:

  • HTML5
  • HTML5 Browsing Contexts and Windows
  • Notifications API

CSS:

  • CSS Basic User Interface
  • CSS Shapes
  • CSS Masking

Browser and Testing Tools:

  • WebDriver

Accessibility:

  • Accessibility

SemanticWeb:

  • SPARQL Query Language for RDF

위의 규격 이외에도, 여러분이 관심을 갖고 있던 다른 표준 규격을 선택할 수도 있습니다.

You must be able to hand code HTML, JavaScript and CSS to participate. Please bring your own laptop.

Before you begin

Please complete the following in preparation for the event:

  • GitHub Setup
  • Choose your favorite text editor or IDE
  • Install all major browsers (such as Chrome, Firefox, Internet Explorer, or Opera)
  • Be sure you have testharness.js

If you have problems with setup, experts can help you at the event.

Prior to attending, you will want to read and become familiar with at least one of the specs so that you have the foundation required to participate.

Here are the specs we will focus on learning and testing:

HTML5:

  • HTML5
  • HTML5 Browsing Contexts and Windows
  • Notifications API

CSS:

  • CSS Basic User Interface
  • CSS Shapes
  • CSS Masking

Browser and Testing Tools:

  • WebDriver

Accessibility:

  • Accessibility

SemanticWeb:

  • SPARQL Query Language for RDF

You may also choose any other W3C spec that interests you.
