The MediaKeyMessageEvent constructor creates a new {{domxref(“MediaKeyMessageEvent”)}} object which creates a new instance of MediaKeyMessageEvent.

Syntax

var mediaKeyMessageEvent = new MediaKeyMessageEvent(typeArg, options)

Parameters

  • typeArg
    • : A {{domxref(“DOMString”)}} containing one of May be one of license-request, license-renewal, license-renewal, or individualization-request.

  • options

    • : Options are as follows:

      • messageType: A developer-defined message type that allows applications to differentiate messages without parsing them.
      • message: An array containing the message generated by the content decryption module.
