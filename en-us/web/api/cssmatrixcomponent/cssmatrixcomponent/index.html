The CSSMatrixComponent() constructor creates a new {{domxref(“CSSMatrixComponent”)}} object representing the matrix() and matrix3d() values of the individual {{CSSXRef(‘transform’)}} property in CSS.

Syntax

var CSSMatrixComponent = new CSSMatrixComponent(matrix[,options]);

Parameters

  • {{domxref(‘CSSMatrixComponent.matrix’,‘matrix’)}}
    • : A 2d or 3d matrix.
