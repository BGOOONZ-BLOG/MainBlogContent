{{APIRef(“DOM Events”)}}

The pageY read-only property of the {{domxref(“MouseEvent”)}} interface returns the Y (vertical) coordinate in pixels of the event relative to the whole document. This property takes into account any vertical scrolling of the page.

Value

A double floating point value.

Examples

let pageY = event.pageY;
