{{APIRef(“URL API”)}}

The values() method of the {{domxref(“URLsearchParams”)}} interface returns an {{jsxref(“Iteration_protocols”,‘iterator’)}} allowing iteration through all values contained in this object. The values are {{domxref(“USVString”)}} objects.

{{availableinworkers}}

Syntax

searchParams.values();

Parameters

None.

Return value

Returns an {{jsxref(“Iteration_protocols”,“iterator”)}}.

Examples

// Create a test URLSearchParams object
var searchParams = new URLSearchParams("key1=value1&key2=value2");

// Display the values
for(var value of searchParams.values()) {
  console.log(value);
}

The result is:

value1
value2
