The NDEFReader() constructor of the {{domxref(“NDEFReader”)}} interface returns a new NDEFReader object, which is used to read NDEF messages from compatible NFC devices, e.g. NDEF tags, within the reader’s magnetic induction field.

Syntax

let reader = new NDEFReader();

Parameters

None.

Return value

A new {{DOMxRef(“NDEFReader”)}}.
