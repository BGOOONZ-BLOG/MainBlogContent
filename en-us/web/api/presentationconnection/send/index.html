{{APIRef(“Presentation”)}}

The send() method of the {{domxref(“PresentationConnection”)}} interface tells a controlling browsing context to send binary or text data to a presenting browsing context.

Syntax

PresentationConnection.send(data);

Parameters

  • data

    • : The data to send to the presentation context. It will be one of the following:

      • {{jsxref(“String”)}}
      • {{domxref(“Blob”)}}
      • {{jsxref(“Array”)}}

Return value

undefined.
