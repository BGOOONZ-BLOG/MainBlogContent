The CanvasRenderingContext2D.canvas property, part of the Canvas API, is a read-only reference to the {{domxref(“HTMLCanvasElement”)}} object that is associated with a given context. It might be {{jsxref(“null”)}} if there is no associated {{HTMLElement(“canvas”)}} element.

Syntax

ctx.canvas;

Examples

Given this {{HTMLElement(“canvas”)}} element:

<canvas id="canvas"></canvas>

… you can get a reference to the canvas element within the CanvasRenderingContext2D by using the canvas property:

var canvas = document.getElementById('canvas');
var ctx = canvas.getContext('2d');
ctx.canvas // HTMLCanvasElement
