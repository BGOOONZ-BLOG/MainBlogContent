{{APIRef(“Canvas API”)}}

The HTMLCanvasElement.height property is a positive integer reflecting the {{htmlattrxref(“height”, “canvas”)}} HTML attribute of the {{HTMLElement(“canvas”)}} element interpreted in CSS pixels. When the attribute is not specified, or if it is set to an invalid value, like a negative, the default value of 150 is used.

This is one of the two properties, the other being {{domxref(“HTMLCanvasElement.width”)}}, that controls the size of the canvas.

Syntax

var pxl = canvas.height;
canvas.height = pxl;

Examples

Given this {{HTMLElement(“canvas”)}} element:

<canvas id="canvas" width="300" height="300"></canvas>

You can get the height of the canvas with the following code:

var canvas = document.getElementById('canvas');
console.log(canvas.height); // 300
