{{APIRef(“File API”)}}

The abort method aborts the read operation. Upon return, the {{domxref(“FileReader.readyState”,“readyState”)}} will be DONE.

Syntax

instanceOfFileReader.abort();

Exceptions

  • DOM_FILE_ABORT_ERR
    • : Thrown when abort is called while no read operation is in progress (that is, the state isn’t LOADING).
