The FormData.keys() method returns an {{jsxref(“Iteration_protocols”,‘iterator’)}} allowing to go through all keys contained in this object. The keys are {{domxref(“USVString”)}} objects.

Note: This method is available in Web Workers.

Syntax

formData.keys();

Return value

Returns an {{jsxref(“Iteration_protocols”,“iterator”)}}.

Example

// Create a test FormData object
var formData = new FormData();
formData.append('key1', 'value1');
formData.append('key2', 'value2');

// Display the keys
for (var key of formData.keys()) {
   console.log(key);
}

The result is:

key1
key2
