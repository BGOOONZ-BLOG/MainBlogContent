The HTMLTableElement.border property represents the border width of the {{HtmlElement(“table”)}} element.

Syntax

HTMLTableElement.border = border;
var border = HTMLTableElement.border;
  • border is a string representing the width of the border in pixels.

Example

// Set the width of a table border to 2 pixels
var t = document.getElementById('TableA');
t.border = '2';

Specifications

W3C DOM 2 HTML Specification HTMLTableElement .border.

This attribute is deprecated in HTML 4.0.
