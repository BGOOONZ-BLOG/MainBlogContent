{{APIRef(“DOM”)}}

The hasChildNodes() method of the {{domxref(“Node”)}} interface returns a boolean value indicating whether the given {{domxref(“Node”)}} has child nodes or not.

Syntax

hasChildNodes();

Paramenters

None.

Return value

A boolean value that is true if the node has child nodes, and false otherwise.

Example

let foo = document.getElementById('foo');

if (foo.hasChildNodes()) {
  // Do something with 'foo.childNodes'
}
