The size() method of the {{domxref(“CountQueuingStrategy”)}} interface always returns 1, so that the total queue size is a count of the number of chunks in the queue.

Syntax

var size = countQueuingStrategy.size();

Parameters

None.

Return value

1.

Examples

const queuingStrategy = new CountQueuingStrategy({ highWaterMark: 1 });

const writableStream = new WritableStream({
  // Implement the sink
  write(chunk) {
    ...
  },
  close() {
    ...
  },
  abort(err) {
    console.log("Sink error:", err);
  }
}, queuingStrategy);

var size = queuingStrategy.size();
