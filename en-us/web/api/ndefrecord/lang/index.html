The lang property of the {{DOMxRef(“NDEFRecord”)}} interface returns the language of a textual payload, or null if one was not supplied.

The record might be missing a language tag, for example, if the recorded information is not locale-specific.

Syntax

NDEFRecord.lang

Value

A {{DOMxRef(“USVString”)}}.
