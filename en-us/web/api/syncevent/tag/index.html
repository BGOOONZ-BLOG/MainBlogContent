The SyncEvent.tag read-only property of the {{domxref(“SyncEvent”)}} interface returns the developer-defined identifier for this SyncEvent. This is the value passed in the tag parameter of the {{domxref(“SyncEvent.SyncEvent”,“SyncEvent()”)}} constructor.

Syntax

var tag = SyncEvent.tag

Value

The developer-defined identifier for this SyncEvent.
