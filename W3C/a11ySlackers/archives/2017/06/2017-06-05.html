    [karlgroves] @pstonier The biggest challenge with dataviz is for a user to gain the same immediate understanding of the chart that a non-disabled user gets by seeing the chart. While a data table affords the user with the ability to explore the data on their own, it doesn’t give that “at a glance” understanding. That said, the tabular data is still important to have.

    I’ve never seen/ used Quill or Automated Insights, but it does seem as though the missing link is some mechanism of providing an interpretation of the data.
