The wakeLock read-only property returns a {{DOMxRef(“WakeLock”)}} interface which allows a document to acquire a screen wake lock. While a screen wake lock is active, the user agent will try to prevent the device from dimming the screen, turning it off completely, or showing a screensaver.

Syntax

const wakeLock = navigator.wakeLock;
