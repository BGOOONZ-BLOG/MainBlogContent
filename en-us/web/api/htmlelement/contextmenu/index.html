The HTMLElement.contextMenu property refers to the context menu assigned to an element using the {{htmlattrxref(“contextmenu”)}} attribute. The menu itself is created using the {{HTMLElement(“menu”)}} element.

Syntax

var elementContextMenu = element.contextMenu;

Example

var contextMenu = document.getElementById("element").contextMenu;

// Change the label of the first menu entry
contextMenu.firstElementChild.label = "New label";
