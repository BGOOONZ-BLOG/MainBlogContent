{{APIRef(“Screen Orientation”)}}

The onchange property of the {{domxref(“ScreenOrientation”)}} is an event handler fired whenever is the event handler called when the screen changes orientation.

Syntax

screen.orientation.addEventListener('change', function(e) { /* ... */ })
screen.orientation.onchange = function(e) { /* ... */ }
