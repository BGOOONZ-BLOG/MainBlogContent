The appendStream() method of the {{domxref(“SourceBuffer”)}} interface appends media segment data from a ReadableStream object to the SourceBuffer.

Syntax

sourceBuffer.appendStream(stream, maxSize);

Parameters

  • stream
    • : The {{domxref(“ReadableStream”)}} that is the source of the media segment data you want to append to the SourceBuffer.
  • maxSize
    • : An unsigned long value indicating the maximum number of bytes that can be appended in this operation.

Return value

{{jsxref(‘undefined’)}}.

Exceptions

None.
