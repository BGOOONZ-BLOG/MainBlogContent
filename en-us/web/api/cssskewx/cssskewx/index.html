The CSSSkewX() constructor creates a new {{domxref(“CSSSkewX”)}} object which represents the skewX() value of the individual {{CSSXRef(‘transform’)}} property in CSS.

Syntax

var CSSSkewX = new CSSSkewX(ax);

Parameters

  • {{domxref(‘CSSSkewx.ax’,‘ax’)}}
    • : A value for the ax angle of the {{domxref(‘CSSSkewX’)}} object to be constructed. This must be a {{domxref(‘CSSNumericValue’)}}.
