paintWorklet is a static, read-only property of the {{DOMxRef(“CSS”)}} interface that provides access to the {{DOMxRef(“PaintWorklet”)}}, which programmatically generates an image where a CSS property expects a file.

Syntax

var worklet = CSS.paintWorklet;

Value

The {{DOMxRef(‘PaintWorklet’)}} object.

Examples

The following example demonstrates loading a {{DOMxRef(‘PaintWorklet’)}} from its js file and does so by feature detection.

<script>
  if ('paintWorklet' in CSS) {
    CSS.paintWorklet.addModule('checkerboard.js');
  }
</script>
