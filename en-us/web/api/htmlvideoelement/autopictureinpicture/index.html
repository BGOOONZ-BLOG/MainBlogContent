The {{domxref(“HTMLVideoElement”)}} autoPictureInPicture property reflects the HTML attribute indicating whether the video should enter or leave picture-in-picture mode automatically.

Syntax

autoPictureInPicture = htmlVideoElement.autoPictureInPicture;

Value

A boolean value that is true if the video should enter or leave picture-in-picture mode automatically when changing tab and/or application.
