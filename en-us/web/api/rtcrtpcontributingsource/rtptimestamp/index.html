The read-only rtpTimestamp property of the {{domxref(“RTCRtpContributingSource”)}} dictionary contains a {{domxref(“DOMHighResTimeStamp”)}} indicating the source-generated time at which the media contained int he packet was first sampled or obtained.

Syntax

let rtpTimestamp = RTCRtpContributingSource.rtpTimestamp

Value

An integer value specifiying a source-generated timestamp indicating the time at which the media in this packet, scheduled for play out at the time indicated by {{domxref(“RTCRtpContributingSource.timestamp”, “timestamp”)}}, was initially sampled or generated. This value may be useful for sequencing and synchronization purposes.
